# 📊 SuperStore Time Series Clustering Dashboard

Dashboard ini dibuat menggunakan Streamlit untuk analisis clustering data time series menggunakan:

- DTW (Dynamic Time Warping)
- CID-DTW (Complexity Invariant Distance)
- Hierarchical Clustering (Ward Linkage)

## 🔍 Fitur
- Visualisasi time series
- Analisis statistik (CV, CE, ACF, slope)
- Clustering interaktif
- Dashboard berbasis Streamlit

## 🚀 Cara Menjalankan
```bash
streamlit run app.py
